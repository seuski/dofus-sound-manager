Comment ça marche

Simple à utiliser, aucune configuration technique requise.

1

Installez

Téléchargez et lancez l'installeur. Tout se configure automatiquement.

2

Coupez la musique Dofus

Dans Dofus, allez dans Options → Son et mettez le curseur Musique à 0. Les effets sonores restent actifs.

3

Choisissez votre version

Sélectionnez 1.29 ou 2.0 selon vos préférences nostalgiques.

4

Activez

Cliquez sur Activer, entrez votre MapId actuel (/mapid dans le chat Dofus) et profitez.

5

Jouez !

La musique change automatiquement selon la zone où vous vous trouvez. Combats, donjons, zones — tout est géré.

FAQ

Questions fréquentes.

Comment couper la musique de Dofus sans couper les effets sonores ? ▼
Dans Dofus, ouvrez le menu Options (touche Échap) → Son → déplacez uniquement le curseur Musique à 0. Les curseurs Effets et Ambiance peuvent rester comme vous le souhaitez.
Avec quelle version de Dofus est-il compatible ? ▼
Dofus Sound Manager est compatible avec Dofus 3 (Unity). Il n'est pas compatible avec Dofus Rétro.
Pourquoi Windows affiche un avertissement de sécurité ? ▼
Ce message apparaît pour tous les logiciels indépendants qui ne possèdent pas de certificat de signature payant. Cliquez sur "Informations complémentaires" puis "Exécuter quand même". Le logiciel est entièrement sûr et open source.
Comment mettre à jour le logiciel ? ▼
Le programme vérifie automatiquement les mises à jour à chaque démarrage. Si une nouvelle version est disponible, une notification apparaît avec un bouton de téléchargement direct.
Qu'est-ce que le MapId et comment le trouver ? ▼
Le MapId est l'identifiant unique de la map où vous vous trouvez. Pour le connaître, tapez /mapid dans le chat de Dofus. Ce numéro est demandé une seule fois au démarrage pour calibrer la détection.
Le logiciel peut-il me faire bannir ? ▼
Dofus Sound Manager lit uniquement la mémoire du jeu en lecture seule pour détecter la map actuelle. Il ne modifie aucun fichier du jeu et n'interagit pas avec les serveurs d'Ankama. Le risque est donc extrêmement faible, mais l'utilisation reste à vos propres risques.
Comment ajouter mes propres musiques ? ▼
Sélectionnez la version Perso dans l'interface et indiquez un dossier contenant vos fichiers MP3 nommés par zone (ex: amakna.mp3, bonta.mp3, frigost.mp3...). La liste complète des noms de fichiers est disponible sur la page GitHub du projet.
Est-ce que ça fonctionne sur Mac ou Linux ? ▼
Non, Dofus Sound Manager est actuellement Windows uniquement. La lecture mémoire utilisée pour détecter les maps repose sur des API Windows spécifiques.
